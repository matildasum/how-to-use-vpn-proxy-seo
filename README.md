## FAQ

**1) How do I verify it’s working after I connect?**  
Connect, then check your public IP and location in a browser. It should match the selected server region.

**2) What should I turn on first: Kill Switch or connect?**  
Turn on **Kill Switch** first. Then connect. That way you don’t leak traffic during setup or reconnects.

**3) Will this slow down my connection?**  
Any VPN/proxy adds overhead. Pick a nearby server for the best speed, or switch servers if latency spikes.

**4) Does it keep logs of what I do?**  
It follows a **No-Logs policy**. Your browsing activity isn’t stored as usage history.
